# TechTrivia

AI-powered voice interview practice for technical roles. Have real-time spoken conversations with an AI interviewer that asks follow-up questions and provides detailed feedback.

## What It Does

Practice technical interviews through natural voice conversations with an AI interviewer.

**Interview Types:**
- Tech Trivia (Java, Python, JavaScript concepts)
- System Design (architecture and scalability)

**How It Works:**
1. Speak your answer
2. AI asks follow-up questions
3. Get transcript + detailed feedback

No typing, no code editor - just talking through concepts like a real interview. We know, in about 3 years you won't even be writing code anyways. :)

## Tech Stack

**Frontend**: React + Vite + TypeScript + CSS Modules
**Backend**: Express + TypeScript + Prisma
**Database**: PostgreSQL
**AI**: Claude API (conversation) + OpenAI TTS (voice)
**Auth**: Clerk
**Payments**: Stripe

Built to simulate real interview pressure without booking expensive mock sessions.
