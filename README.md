# techpad.app

AI-powered voice interview practice for technical roles. Have real-time spoken conversations with an AI interviewer that asks follow-up questions and provides detailed feedback.

What It Does
Practice technical interviews through natural voice conversations with an AI interviewer.

Interview Types:

- System Design
- Low-level Concepts
- Cloud Architecture
- Data Structures
- IT Administration
- *more to come...*

How It Works:

- Select an interview preset
- Speak, draw, code your answer
- AI asks adaptive followups to what you've just said
- Get transcript + detailed feedback
